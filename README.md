# Active-Directory-Lab
A hands-on Active Directory lab documenting the full journey from environment setup to real world exploitation of ADCS misconfigurations.
![Image Alt](https://raw.githubusercontent.com/kaveeshanirmal141/Active-Directory-Lab/refs/heads/main/AD%20art%20(Optional).jpeg)
> ⚠️ This lab is built for educational purposes to understand how Active Directory environments can be misconfigured and exploited, and how to defend against such scenarios.
>
> This project covers:
- Building a multi-machine AD environment (DC, CA, Client)
- Debugging real-world issues (DNS, IPv6 vs IPv4, AD services)
- Understanding Active Directory Certificate Services (ADCS)
- Creating and exploiting ESC1 misconfiguration and more...
- Performing certificate-based authentication (PKINIT)
- Achieving Domain Admin access without passwords

Includes detailed documentation of:
- Challenges and debugging process
- Misconfigurations and their impact
- Exploitation workflow with proof of access
- Key insights from an attacker and defender perspective

This repository is not just a lab — it is a structured breakdown of how Active Directory environments are built, broken, and understood.
